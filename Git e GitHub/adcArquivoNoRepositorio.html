Como adicionar arquivos no repositórios?

Resumo:

  • Criamos um diretório no guiBash com o MKDIR.

  • Iniciamos um projeto GIT com o git init.

  • Criamos um repositório no gitHub.

  • Por ser privado este exemplo, adicionamos um token ou uma chave ssh.
    **Atenção ao processo.

  • Configuramos nosso projeto git no repositório do github com o comando: "git remote add origin [link do repositório gitHUB].

  • Geramos nossos arquivos e agora vamos realizar o "upload" dos mesmos. Usamos o comando: git add {nome do arquivo com sua extensão}
    *Neste momento o seu arquivo esta na zona de preparo para o proximo passo que é o commit.
    **Um atalho para adicionar todos os arquivos é usar o "ponto" no lugar do nome do arquivo.

  • Com o comando: git commit -m"sua mensagem aqui"

Atenção!Caso você veja um arquivo dentro de uma pasta vazia em algum respositório chamado .gitkeep é somente para que o git reconheça esse diretório e também faça o upload dele. Obviamente, o git não reconhece pastas vazias.
