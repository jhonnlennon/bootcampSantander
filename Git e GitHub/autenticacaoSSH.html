Como é feita a autenticação via SSH?

É um protocolo onde o computador local e o servidor, através de uma conexão criptografada se conectam.


Analisando a documentação

Inicialmente para saber e acessar uma chave ssh devemos usar o seguinte comando: ls -al ~/.ssh veja mais sobre o comando aqui..

Caso retorne "cannot access... No such file or dir...." Justamente porque não tem nenhum item.

Gerando uma chave SSH

Para estarmos gerando uma nova chave SSH devemos seguir os seguintes comandos: Clique aqui.
Em seguida, ele vai pedir para indicar onde vai salvar a chave. Deixe sempre padrão. (se possível.)
Logo após, digite uma frase secreta, como senha.

Guardando a chave privada

Lembrando que geramos uma chave pública e uma privada. Após gerar a chave SSH, devemos adicionar a chave privada ao ssh-agente (fica no computador local), para que seja corretamente armazenada. Vamos executar o ssh-agente.. Veja abaixo.

Clique aqui para visualizar o comando para executar o SSH-AGENT. Em seguida devemos adicionar nossa chave privada. Veja como:
Deve digitar o seguinte comando:
ssh-add ~/.ssh/id_ed25519
Em seguida, digite sua frase secreta.

Adicionado a chave privada no ssh-agent, devemos adicionar a chave pública no GitHUB. Veja abaixo.

Guardando a chave pública

Como dito anteriormente, devemos agora adicionar nossa chave pública no GitHUB.
A ideia é simples, a chave privada(computador local) vai se comunicar com a pública (GitHUB)
Por isso, cada uma deve estar em seu devido local.

Para acessar SSH key no GitHUB, devemos navegar até a opção Settings (canto superior direito)...
Em seguida selecionar "SSH and GPG Keys"..

Devemos adicionar um título, para que possamos saber de onde é esta chave.

Também selecionar a chave de "Authentication Key".
**A opção "Signing Key" é para adicionar assinatura (validação) aos seus commit.

Para acessar nossa chave, devemos acessar o local "home" onde estão nossas chaves, o comando é: cd ~/.shh

  • "cd" siginifica change direct, trocar de diretório.
  • "~/" diretório (endereço)
  • ".shh" pasta oculta onde ficam as chaves.


Utilize a chave com final .pub, depois de usar um LS para listar o conteúdo da pasta.
**Para visualizar, usar o comando cat nomedachave.pub

Atenção, para clonar usando o ssh, usar no GitBASH, já dentro do diretório onde quer fazer o clone e usar, git clone e o link do ssh. Lembrar de colocar a palavra secreta.
