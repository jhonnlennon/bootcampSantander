Como é feita a autenticação via SSH?

É um protocolo onde o computador local e o servidor, através de uma conexão criptografada se conectam.

Para acessar devemos navegar até a opção Settings (canto superior direito)...
Em seguida selecionar "SSH and GPG Keys"..
