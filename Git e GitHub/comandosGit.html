Listagem de comandos e suas funções.

  • git clone
      • Utilizado para clonar o repositório.
      *Necessário criar um token ou ssh.
      Para renomear a pasta quando clonado, basta ao final do link adicionar o nome da pasta que quiser.

  • git commit
      • Grava alterações no reposiório.

  • git pull
      • Utilizado para atualizar o seu repositório local, uma vez que tiver outros colaboradores trabalhando em cima do repositório.
      Em tese, ele busca e mescla com o seu para que vc possa atualizar.

  • git push
      • Utilizado para enviar (empurrar) suas alterações para o repositório remoto.

  • git init
      • Transforma a pasta onde esta em um repositório git.

  • git status
      • Usado para mostrar nossa arvore de trabalho e também nossa área de preparação.
      **Arquivos untracked files são arquivos não rastreados, ou seja, aqueles arquivos que o git não conhece.

  • git log
      • Mostra o log dos commits.

  • git restore
      • Utilizado para restaurar um arquivo que fora acidentalmente alterado.

  • git commit --amend -m"sua nova mensagem"
      • Utilizado para alterar a mensagem do seu último commit.

  • git remote add origin [link do repositório gitHUB]
      • Este comando adiciona o repositório local em remoto a partir de um repositório gerado no GitHUB.
  • $ ssh-keygen -t ed25519 -C "seueemail@com.br"
      • O "$" siginifica que é um novo comando no terminal.
      O "-ssh-keygen" responsável por estar gerando nossas chaves.
      O "-t" indica o tipo de algoritimo.
      O "-C" adiciona um comentário, neste caso, o proprietário desta chave.

      1. eval "$(ssh-agent -s)"
          2. Utilizado para executar o ssh-agent.

      2. ssh-add ~/.ssh/id_ed25519
          3. Comando para adicionar uma chave privada ao ssh-agent.

      3. mkdir
          4. Cria um diretório.

      4. touch {nome do arquivo}
          5. Usando para criar um arquivo vazio.

      5. windows + .
          6. open a menu for choise icons or emoji for titles in files .md (markdown)

      6. rm -rf {nome do diretório/arquivo}
          7. vai remover a força todos os arquivos ou diretórios o qual foi selecionado.
