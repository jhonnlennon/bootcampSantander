Listagem de comandos e suas funções.

  • git clone
      • Utilizado para clonar o respositório.
      *Necessário criar um token.

  • git commit
      • Grava alterações no reposiório.

  • git pull
      • Utilizado para atualizar o seu repositório local, uma vez que tiver outros colaboradores trabalhando em cima do respositório.
      Em tese, ele busca e mescla com o seu para que vc possa atualizar.

  • git push
      • Utilizado para enviar (empurrar) suas alterações para o respositório remoto.
