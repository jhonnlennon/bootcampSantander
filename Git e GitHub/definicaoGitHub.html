O que é gitHub?

O gitHub é distindo do GIT (que é sistema para versionamento e controle de códigos), o gitHub é uma plataforma de hospedageria de código para controle de versão com GIT.

É um ambiemte colaborativo já que vamos ter acesso a diversos códigos de outros programadores podendo assim contribuir também.

Ele tem um mascote é que meio humano, meio gato e meio povo que se chama "Octacat".


Foi criado em 2008 por Chris Wanstrath, J. Hyett, Tom Preston-Werner e Scott Chacon.
Em 2018 foi comprado pela Microsoft Corporation por US $7,5 bilhões.
