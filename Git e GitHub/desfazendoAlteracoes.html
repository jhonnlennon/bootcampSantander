Como desfazer algumas alterações.

Caso 1:

  • - Foi realizado um git ini em uma pasta errada.
      • Utilize o comando rm -rf .git, vai remover a força todo o seu conteúdo.

Caso 2:

  • Foi alterado ou removido algum arquivo ou conteúdo.
      • Utilize o comando git restore {nomedoarquivo}

Caso 3:

  • Quero modificar a mensagem do commit.
      • Utilize o comando git commit --amend -m"sua nova mensagem"

Caso 4:

  • Modificando o repositório para a versão específica:

      • - Comando: git reset --sotf

      Primeiro digite o comando: git log para verificar o log dos commits.
      Em seguida, digite o comando: git reset --soft {numero do commit}

      Atenção! Este comando pega todos os arquivos/alterações realizados DEPOIS do commit o qual escolheu e deixa-os na área de preparação para posterior outro commit.


      - Comando: git reset --mixed

      Primeiro digite o comando: git log para verificar o log dos commits.
      Em seguida, digite o comando: git reset --mixed {numero do commit}

      Atenção! Este comando pega todos os arquivos/alterações realizados DEPOIS do commit o qual escolheu e deixa-os como untracked files..


      - Comando: git reset --hard

      Primeiro digite o comando: git log para verificar o log dos commits.
      Em seguida, digite o comando: git reset --hard {numero do commit}

      Atenção! Este comando pega todos os arquivos/alterações realizados DEPOIS do commit o qual escolheu e os ignora e os apaga..

Caso 5:

  • Quero retirar um arquido da minha área de preparação.
      • Utilize o comando git reset {nome do local do arquivo e o nome do arquivo}
      Outro comando: git restore --staged {nome do arquivo junto do caminho onde esta.}
