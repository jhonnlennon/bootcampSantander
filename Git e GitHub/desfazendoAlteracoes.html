Como desfazer algumas alterações.

Caso 1:

  • - Foi realizado um git ini em uma pasta errada.
      • Utilize o comando rm -rf .git, vai remover a força todo o seu conteúdo.

Caso 2:

  • Foi alterado ou removido algum arquivo ou conteúdo.
      • Utilize o comando git restore {nomedoarquivo}

Caso 3:

  • Quero modificar a mensagem do commit.
      • Utilize o comando git commit --amend -m"sua nova mensagem"
