Sobre a TAG IMG

A tag IMG é utilizada para buscar uma imagem (dentrou ou fora) do seu projeto.

Sobre sua sintaxe: < img src=" Aqui dentro, coloque o link. " alt=" Aqui dentro, a descrição da imagem. " >

Importante! O atributo ALT serve também para os leitores do site identificarem o que é esta imagem. No caso, do que se trata.
