Sobre a tag de listas!

Utilizado para listar uma determinada sequencia, utilizamos:

    A Tag "ul" é utilizada quando as ORDENS dos itens não são importantes.
    A Tag "ol" é utilizada quando as ordens dos itens são importantes. São representados por números, letras e algarismos romanos.
  • A tag "li", é sempre um item de uma destas listas acima.
