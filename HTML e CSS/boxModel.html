Sobre Box Model

O que é Box Model?

O navegador, quando realiza a leitura de cada tag HTML, o mesmo identifica como uma caixa retangular. Chamamos isso de BOX MODEL.

O Box Model tem 4 áreas, vide abaixo:

Como funciona cada uma delas?

  • Margin
      • São espaçamentos entre elementos!
  • Border
      • Circunda o Padding e o Content
      Podemos alterar largura e cor.
  • Padding
      • Espaçamento entre o border e content.
  • Content
      • É o que o seu bloco representa, seja um texto ou uma imagem.
