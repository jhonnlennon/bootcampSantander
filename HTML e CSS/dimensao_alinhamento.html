Sobre dimensão e alinhamentos.

  • width
      • Altura que este elemente tem relacionado ao que vem de fora. (tela)
  • Height
      • Largura que este elemente tem relacionado ao que vem de fora. (tela)
  • Max-Width
      • Altura máxima que este elemento pode ter.
  • Max-Height
      • Largura máxima que este elemento pode ter.
  • Margin
      • Coloca espaçamentos entre elementos. Também tem a opção "Auto" para alinhar algum elemento automaticamente.
  • Text align
      • Serve para alinhar os textos.
